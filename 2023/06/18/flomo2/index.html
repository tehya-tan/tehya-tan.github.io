Flomo：趁手的笔记本 (二)

Posted on 2023-06-18 Edited on 2023-06-27 In Tech Views: Disqus:

电脑上做Flomo笔记的快捷方法和读书笔记的录入。

1 前言

这个系列会总结一些我日常使用的flomo功能。

上一篇讲了怎么用flomo做手机笔记，这篇会讲怎么在电脑做flomo笔记和笔记的导入导出（💰都是仅会员可用），包括:

  1. 怎么快速选中网站上的文字发送到flomo笔记？
  2. 怎么导入kindle和微信读书的笔记?
  3. 怎么把flomo笔记同步到Notion？

官网: https://help.flomoapp.com/

下载链接: https://help.flomoapp.com/basic/app.html

2 电脑笔记

2.1 Flomo plus: chrome 添加项

官网介绍：https://help.flomoapp.com/advance/extension/chrome-extension.html

  • 添加之后我们可以在chrome用右键快捷传送选中的文字到flomo

  • 笔记中会自动添加标签和网页链接

设置步骤

  1. 添加chrome添加项 https://chrome.google.com/webstore/detail/flomoplus/kcijjmomofpdcpeiagibojhjifhegepj?hl=zh-CN >

  2. 设置添加项

  3. 输入flomo API 并选择默认的标签

  4. 可以选中文本传送到flomo了🎉

3 读书笔记导入

3.1 Kindle

官网介绍：https://help.flomoapp.com/advance/extension/send2flomo.html#功能介绍

把kindle读书笔记传送flomo的在线网页: https://tit1e.github.io/kindle2Flomo/old/

把kindle读书笔记传送flomo的App下载: https://evolly.one/p/20f2e93b.html

  • 导入kindle的批注之后可以在flomo上方便浏览

  • 如果把设置的阅读标签绑定在每日回顾，flomo会在每日回顾中发送一些过去的阅读批注让你回顾

设置步骤 (在线网页对kindle的批注txt文件摘取传送，app还没用🧎‍♀️)

  1. 上传kindle里的 My Clippings.txt 文件

  2. 填入flomo api

  3. 设置tag，这个tag会被放在书籍名的前面

  4. 选择书籍，可以编辑书名。

  5. 全选或者点击自己想要导入

  6. 设置每日回顾

3.2 微信读书

还是可以借助flomo plus在电脑或者手机上对微信读书的批注进行传送，同样也可以编辑标签和阅读书籍的名字

  • chrome添加项导入

  • 微信小程序导入

    1. 在微信读书上复制所有读书笔记到粘贴板

  1. 打开flomoplus微信小程序，设置api，点击粘贴

5 后记

  • 拖了好久，flomo会员也有2年了。
  • 在信息泛滥的时代，拥有一个趁手的电子笔记本非常重要，它可以帮助我们使用应用程序App来记录信息，而不是依靠人类有限的大脑资源去记忆。<img src="https://github.com/tehya-tan/MarkdownImages/blob/master/Images%20061823/Untitled%2021.png?raw=true"  width="60%" >
  • flomo很好地满足了大部分人和我需要的：
    1. 快速捕捉信息，即没有阻碍地快速地保留摘抄、记录想法 ✏️。
    2. 方便整理信息，即能同步到用来写作的电子大脑🧠 💻。
    3. 进行定时回顾，即相信你的笔记会定时浮现出来，帮助你重新消化信息并激发新的想法💡。
